How It Works:
-> generate_password(length) Function:

It generates a password of the specified length using random choices of letters, digits, and punctuation.
-> User Input:

The user is prompted to input the desired password length.
-> Validation:

The code checks that the user has entered a valid number and ensures that the length is at least 1.
Password Generation:

-> The password is generated and displayed.
